# Contribuir a Chat Tres Valles

¡Gracias por tu interés en contribuir! Este es un proyecto de portafolio profesional.

## Cómo contribuir
1. Haz un Fork del repositorio.
2. Crea una rama para tu característica (`git checkout -b feature/AmazingFeature`).
3. Haz tus cambios y confírmalos (`git commit -m 'Add some AmazingFeature'`).
4. Sube tus cambios (`git push origin feature/AmazingFeature`).
5. Abre un Pull Request.
