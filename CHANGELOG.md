# Changelog

Todos los cambios notables en el proyecto "Chat Tres Valles" se documentarán en este archivo.

## [1.0.0] - 2025-12-30
### Añadido
- Sistema base con Flask y SocketIO.
- Base de datos SQLite (chat_history.db) optimizada.
- Soporte para emojis modernos (Picmo).
- Sistema de salas geolocalizadas.
- Identidad visual con avatares dinámicos (DiceBear).
- Interfaz responsiva para móviles y escritorio.
- Documentación inicial (README, LICENSE, CONTRIBUTING).
